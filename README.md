# Cookiecutter My template

My first attempt at a cookiecutter template. 

Adapted from https://github.com/frankie567/cookiecutter-hipster-pypackage

Includes:
- versioneer
- meta.yaml for building conda package
- mypy.ini
- setup.cfg
- setup.py
- pylama.ini
- pyproject.toml
- .gitignore
- tests
- docs via sphinx
